# Benim Mario Tarzı Oyunum

Bu, Phaser.js kütüphanesi kullanılarak yapılmış, kişiselleştirilmiş bir platform oyunudur.

## Karakterler:
* **Prens:** Siyah pelerinli kahraman (erkek arkadaşım).
* **Prenses:** Kulede mahsur kalmış (ben).

## Kontroller:
* **Ok Tuşları:** Hareket etme, Zıplama ve Eğilme.

*Bu oyun henüz geliştirme aşamasındadır.*
